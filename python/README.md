# Ігрофікс ПК - Геймерський інтернет-магазин v2.0

## 🎮 Опис

Сучасний інтернет-магазин комп'ютерної техніки для геймерів з українською локалізацією. 
Версія 2.0 з покращеними анімаціями, sound effects та ігровим дизайном.

### Основні функції:

- 🛒 **Каталог товарів** - процесори, відеокарти, ОЗП, накопичувачі, корпуси, блоки живлення
- ⚙️ **Конструктор ПК** - збирання власної конфігурації з автоматичною перевіркою сумісності
- 🛍️ **Кошик** - додавання, видалення, оновлення товарів
- 👤 **Профіль користувача** - збережені конфігурації та історія замовлень
- 🛠️ **Адмінпанель** - керування товарами, категоріями, замовленнями

### ✨ Нові функції v2.0:

- ⚡ **Sound Effects** - звуки блискавки при наведенні та кліку
- 🎨 **Покращені анімації** - glow, pulse, lightning effects
- 🖱️ **Кастомний курсор** - змінюється при наведенні на інтерактивні елементи
- 🔊 **Перемикач звуку** - кнопка для ввімкнення/вимкнення звуку
- 📱 **Адаптивний дизайн** - працює на всіх пристроях

## 🚀 Запуск

### Варіант 1: Автоматичний запуск

```bash
run.bat
```

### Варіант 2: Вручну

```bash
# Встановлення залежностей
py -3.13 -m pip install -r requirements.txt

# Запуск сервера
py -3.13 app.py
```

Сервер запуститься на http://localhost:5000

```
project/
├── app.py              # Головний Flask додаток
├── config.py           # Конфігурація
├── models.py           # Моделі бази даних
├── forms.py            # Форми Flask-WTF

│       └── main.js         # Головний JavaScript (~450 рядків)
└── templates/
    ├── base.html           # Базовий шаблон
    ├── index.html          # Головна сторінка
    ├── catalog.html        # Каталог товарів
    ├── product.html        # Сторінка товару
    ├── builder.html        # Конструктор ПК
    ├── cart.html           # Кошик
    ├── checkout.html       # Оформлення замовлення
```

## 🛠️ Технології

- **Backend:** Python 3.13 + Flask
- **База даних:** SQLite
- **ORM:** SQLAlchemy
- **Форми:** Flask-WTF
- **Авторизація:** Flask-Login
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Аудіо:** Web Audio API
